## 프로젝트 개요
Python과 Pygame을 사용하여 제작한 주식 거래 시뮬레이션 게임입니다.  
사용자는 가상의 화폐를 이용해 주식을 사고팔며, 총 자산을 관리할 수 있습니다.  
화폐 종류에는 원, 코인, 금, 스탁 등이 있으며, 상점과 교환소를 통해 거래할 수 있습니다.

---

## 주요 기능

- **실시간 UI 렌더링**
  - Pygame 기반의 직관적인 인터페이스
  - 화폐 버튼 및 종목 리스트 표시
  - 총 보유 자산 및 화폐별 현금 패널
  - 선택된 종목의 차트 표시

- **거래 기능**
  - 주식 구매/판매
  - 수량 조절 버튼(-, +, %, 등)
  - 총액 계산 및 거래 가능 여부 표시

- **모달 창**
  - 상점 및 교환소 모달
  - ESC 키로 모달 닫기 가능

- **보유 종목 관리**
  - 보유 종목 리스트 및 수량 표시
  - 변동폭(손익률) 색상 표시
  - 스크롤 기능 지원

- **숫자 포매팅**
  - 억, 조, 경 단위로 큰 숫자 표시
  - 천 단위 콤마 적용
